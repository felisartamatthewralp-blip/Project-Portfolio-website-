How I built a small task app

Posted: 2025-01-10

This post explains the structure, data model and approach used to build a tiny task manager using vanilla JS and localStorage.

Design

Focus on simplicity and keyboard accessibility. Use semantic markup and aria attributes where appropriate.

Code

Store tasks as JSON in localStorage. Provide add/remove/update functionality and reuse DOM update function to render list.
